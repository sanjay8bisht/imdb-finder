# imdb movie finder
###### Find the details of any movie listed on imdb (for developers)

> Clone the project

```
	cd imdb-finder
	python imdb.py movie+name [year] [open]
```

###### movie name = movie+name+:+if+have+sub+title
###### [] = optional
###### [year] = specify year of the movie
###### [open] = open imdb link in browser
